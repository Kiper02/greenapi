# GREEN-API Тестовый клиент

Веб-интерфейс для тестирования методов GREEN-API: получение настроек и состояния инстанса, отправка текстовых сообщений и файлов по URL.

## Функционал

- Получение настроек инстанса (`getSettings`)
- Получение состояния инстанса (`getStateInstance`)
- Отправка текстового сообщения (`sendMessage`)
- Отправка файла по публичной URL-ссылке (`sendFileByUrl`)

Все параметры подключения (`idInstance`, `apiTokenInstance`) вводятся на самой странице. Ответы от API отображаются в соответствующих полях.

## Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone <url-вашего-репозитория>
   cd greenapi-test
    ```   
2. **Установите зависимости**
    ```bash
    go mod tidy 
   ```

3. **Запустите сервер**
    ```bash
    go run main.go
    ```

4. **Откройте в браузере http://localhost:8080**

## Использование

Введите idInstance и apiTokenInstance из вашего аккаунта GREEN-API.
Нажмите кнопку `Выполнить getSettings` или `getStateInstance` – ответ появится в соответствующем поле.

    Для отправки сообщения:
        Укажите chatId в формате 79001234567@c.us (для личного чата) или 123456789@g.us (для группы).
        Напишите текст сообщения.
        Нажмите Отправить.
    Для отправки файла:
        Укажите chatId.
        Вставьте прямую ссылку на файл (URL).
        При необходимости укажите имя файла и подпись.
        Нажмите Отправить файл по URL.
